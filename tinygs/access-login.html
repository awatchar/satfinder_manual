2) ขั้นตอนเข้าหน้าเว็บไซต์และลงชื่อเข้าใช้

หมายเหตุ: หากจุดประสงค์คือ “สังเกตการณ์/วิเคราะห์ข้อมูล” โดยทั่วไปสามารถทำได้แม้ไม่ลงชื่อเข้าใช้ (ขึ้นกับนโยบายการเข้าถึงของ Dashboard ในเวลานั้น)

กรณี A: เข้าใช้งานเพื่อดูและวิเคราะห์ข้อมูล (ไม่ต้องล็อกอิน)

  1. เปิดเว็บเบราว์เซอร์ (แนะนำ Chrome/Edge รุ่นปัจจุบัน)
  2. ไปที่หน้า TinyGS Dashboard
  3. ตรวจสอบว่าเห็นหน้า Dashboard และองค์ประกอบหลัก เช่น แผนที่, แถบเมนู, รายการข้อมูล
  4. เริ่มจากเลือกดาวเทียมหรือมุมมองข้อมูลที่ต้องการ

กรณี B: ลงชื่อเข้าใช้ (เพื่อใช้งานความสามารถเพิ่มเติม)

  1. เข้าหน้า TinyGS Dashboard
  2. มองหาปุ่ม/เมนู Sign in / Login / Account
  3. เลือกวิธีลงชื่อเข้าใช้ตามที่ระบบรองรับ
  4. ตรวจสอบว่า UI แสดงชื่อผู้ใช้/สถานะบัญชีหรือเมนูเพิ่มขึ้น

ข้อควรระวัง (Caution)

  • หากใช้บัญชีโรงเรียน/บัญชีโครงการ ให้ใช้รหัสผ่านไม่ซ้ำและไม่แชร์ในที่สาธารณะ
  • ไม่บันทึกรหัสผ่านในเครื่องสาธารณะ
  • แยกบัญชีสาธิตสำหรับห้องเรียน ออกจากบัญชีที่ใช้ตั้งค่าระบบจริง
