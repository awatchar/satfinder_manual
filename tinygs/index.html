คู่มือ TinyGS สำหรับครู (SatFinder Manual)

TinyGS เป็น Dashboard สำหรับติดตามสถานีภาคพื้นของโรงเรียนและการรับสัญญาณจากดาวเทียมในสถานการณ์จริง

คู่มือนี้เน้นการอ่านสถานะ/กราฟสำคัญ และการวางแผนกิจกรรมในชั้นเรียนให้เชื่อมกับข้อมูลจริงได้ทันที โดยไม่ต้องเริ่มจากทฤษฎียาว

สำหรับการใช้งานพื้นฐานของครู สามารถเรียนรู้ได้โดยไม่ต้อง Login (รายละเอียด Login อยู่บทที่ 2)

เริ่มบทที่ 1 เริ่มแบบเร็วที่บทที่ 4

เลือกเส้นทางการเรียนรู้

ครูใหม่ (อ่านตามลำดับ)

  • ไล่จากบทที่ 1 → 7 ตามลำดับ
  • เหมาะกับครูที่ต้องการเห็นภาพรวมครบก่อนสอน
  • เริ่มจากพื้นฐานจนถึงการวิเคราะห์ telemetry ข้ามสถานี

สอนเร็วใน 1 คาบ

  • เริ่มที่บทที่ 4: สถานีและ Live Tracking
  • ต่อบทที่ 5: Pass/Timeline/Sky View/Coverage
  • ปิดท้ายบทที่ 6: อ่าน Packet และคุณภาพสัญญาณ

บทเรียน TinyGS (7 บท)

  1. บทที่ 1: TinyGS คืออะไร และใช้ทำอะไรในงานการเรียนการสอน
    สิ่งที่ได้: เข้าใจภาพรวม TinyGS และบทบาทในชั้นเรียน
  2. บทที่ 2: ขั้นตอนเข้าหน้าเว็บไซต์และลงชื่อเข้าใช้
    สิ่งที่ได้: แยกกรณีใช้งานแบบไม่ล็อกอินกับการล็อกอินสำหรับผู้เรียนระดับสูง
  3. บทที่ 3: คำอธิบายเมนูหลักที่ควรรู้ก่อนเริ่มใช้งานจริง
    สิ่งที่ได้: ใช้งานเมนูหลักและค้นหาสถานีโรงเรียนได้ถูกต้อง
  4. บทที่ 4: สถานะ TinyGS Station และ Live Tracking (2D Map/3D Globe)
    สิ่งที่ได้: อ่านสุขภาพสถานีและใช้ 2D/3D วางแผนสังเกตการณ์
  5. บทที่ 5: การอ่าน Pass, Timeline, Sky View และ Coverage Map
    สิ่งที่ได้: เลือกช่วงเวลาที่เหมาะและเชื่อมโยงทิศ/มุมกับโอกาสรับสัญญาณ
  6. บทที่ 6: การอ่านรายการ Packet บน TinyGS Dashboard (RSSI/SNR/Doppler/CRC)
    สิ่งที่ได้: ประเมินคุณภาพการรับจากตัวชี้วัดหลักใน Packet list
  7. บทที่ 7: วิเคราะห์ Packet Telemetry และ Receiving Stations
    สิ่งที่ได้: อ่าน telemetry และเปรียบเทียบคุณภาพการรับข้ามหลายสถานี
⚠️ ข้อควรรู้ก่อนเริ่ม
  • Packets จะมาเป็นช่วงตามการผ่านของดาวเทียม ไม่ได้ต่อเนื่องตลอดเวลา
  • ครูสามารถเรียนและสอนพื้นฐานได้โดยไม่ต้อง Login
  • หากยังไม่เห็นแพ็กเก็ตใหม่ ให้ใช้บทที่ 5 วางแผนเวลา pass ก่อนทดลองอีกครั้ง
โครงการส่งเสริมการเรียนรู้ทางด้านโทรคมนาคมในโรงเรียนทั่วประเทศ (SatFinder)
