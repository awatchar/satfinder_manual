TinyGS Dashboard (สำหรับครูผู้สอน)

คู่มือย่อยสำหรับครู เน้นการใช้งานจริงในห้องเรียน โดยเรียงลำดับจากภาพรวม → เข้าถึงระบบ → เข้าใจเมนูหลัก → อ่านสถานะสถานีและ Live Tracking

เริ่มบทที่ 01

ครูจะได้เรียนรู้อะไร

  • บทบาทของ TinyGS ในการสอนด้านโทรคมนาคมและอวกาศ
  • วิธีเข้าหน้า Dashboard และแนวทางใช้งานแบบไม่ล็อกอิน/ล็อกอิน
  • ความหมายของเมนูหลักเพื่อออกแบบกิจกรรมในคาบเรียน
  • การอ่านสถานะ Station และการใช้ Live Tracking แบบ 2D/3D เพื่อวางแผนการสังเกตการณ์

บทเรียน TinyGS (6 บท)

  1. บทที่ 1: TinyGS คืออะไร และใช้ทำอะไรในงานการเรียนการสอน
  2. บทที่ 2: ขั้นตอนเข้าหน้าเว็บไซต์และลงชื่อเข้าใช้
  3. บทที่ 3: คำอธิบายเมนูหลักที่ควรรู้ก่อนเริ่มใช้งานจริง
  4. บทที่ 4: สถานะ TinyGS Station และ Live Tracking (2D Map/3D Globe)
  5. บทที่ 5: การอ่าน Pass, Timeline, Sky View และ Coverage Map
  6. บทที่ 6: การอ่านรายการ Packet บน TinyGS Dashboard (RSSI/SNR/Doppler/CRC)

เริ่มต้นใน 10 นาที (Quick Path)

  1. อ่านบทที่ 01 Intro เพื่อเข้าใจภาพรวม TinyGS
  2. ไปบทที่ 02 Access/Login เพื่อเตรียมการใช้งานหน้า Dashboard
  3. ต่อบทที่ 03 Main Menus เพื่อวางแผนการสอนและกิจกรรม
  4. ปิดท้ายบทที่ 04 Station Status & Live Tracking เพื่อประเมินสุขภาพสถานีและวางแผนช่วงรับสัญญาณ
  5. ต่อบทที่ 05 Passes, Timeline, Sky View & Coverage เพื่อเชื่อมโยงการทำนาย pass กับแนวโน้มแพ็กเก็ตระยะยาว
  6. ต่อบทที่ 06 Packet List (RSSI/SNR/Doppler/CRC) เพื่อฝึกวิเคราะห์คุณภาพลิงก์จากข้อมูลรับจริงรายแถว

สิ่งที่ต้องเตรียม

  • เว็บเบราว์เซอร์สมัยใหม่ (Chrome / Edge / Firefox)
  • อินเทอร์เน็ตที่เสถียรสำหรับเปิดหน้า Dashboard

สิ่งที่จะเพิ่มภายหลัง

  • ภาพหน้าจอจริงแต่ละขั้นตอน
  • ใบกิจกรรมและแบบฝึกสำหรับนักเรียน
