3) คำอธิบายเมนูหลักที่ควรรู้ก่อนเริ่มใช้งานจริง

หมายเหตุ: ชื่อเมนูใน Dashboard อาจต่างกันเล็กน้อยตามรุ่น/การปรับ UI แต่แนวคิดของเมนูมักอยู่ในกลุ่มเดียวกัน ด้านล่างคือเมนู/มุมมองที่ครูควรทำความเข้าใจเป็นลำดับแรก

1) Satellite / Satellites

ใช้ค้นหาและเลือกดาวเทียมที่ต้องการติดตาม ครูสามารถให้ผู้เรียนเลือกดาวเทียม 1 ดวงเป็นแกนกิจกรรมทั้งคาบ

2) Pass / Passes / Next Pass

แสดงช่วงเวลาที่ดาวเทียมผ่านและอยู่ในช่วงรับสัญญาณได้ ใช้สอนแนวคิดหน้าต่างเวลาในการสื่อสาร

3) Telemetry / Packets / Frames

แสดงข้อมูลจริงที่รับได้ เช่น timestamp, แหล่งที่มา และ payload ใช้สอนความต่างระหว่างสัญญาณกับข้อมูล

4) Map / Stations / Nodes

แสดงตำแหน่งสถานีภาคพื้นและการกระจายตัวของเครือข่าย เหมาะกับการสอน distributed network

5) Statistics / Activity

ดูจำนวนแพ็กเก็ต อัตราการรับ และแนวโน้มตามเวลา เพื่อต่อยอดกิจกรรมสถิติพื้นฐาน

6) Search / Filter + 7) Account / Settings

กรองข้อมูลตามเงื่อนไข และใช้การตั้งค่าเฉพาะเมื่อจำเป็น โดยควรกำหนดขอบเขตการปรับแต่งให้ชัดเจน
