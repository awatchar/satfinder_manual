02) ขั้นตอนเข้าหน้าเว็บไซต์และลงชื่อเข้าใช้

สรุปบทนี้

บทนี้อธิบายการเข้าสู่ระบบ (Login) ของ TinyGS และข้อควรพิจารณาก่อนใช้งานจริง ในหลายกรณีครูสามารถเข้าดู Dashboard เพื่อวิเคราะห์ข้อมูลได้โดยไม่ต้องล็อกอิน

⚠️ CAUTION
การ Login นี้เหมาะสำหรับนักเรียนระดับ Advance เท่านั้น
  • ครูต้องเป็นผู้ส่ง URL สำหรับเข้าสู่ระบบให้ผู้เรียน โดย URL นี้ได้จาก Telegram Bot ตอนสร้าง/ตั้งค่า TinyGS Station
  • หากไม่จำเป็น แนะนำให้ข้ามขั้นตอน Login และใช้การดูข้อมูลแบบไม่ล็อกอินแทน
  • ข้อควรระวัง: อย่าเผยแพร่ URL สำหรับเข้าสู่ระบบต่อสาธารณะ

ขั้นตอนแนะนำ

  1. เปิดเว็บเบราว์เซอร์ (Chrome/Edge/Firefox รุ่นปัจจุบัน)
  2. เข้าสู่หน้า TinyGS Dashboard
  3. ตรวจสอบองค์ประกอบหลัก: เมนู, มุมมองข้อมูล, พื้นที่ค้นหา
  4. กรณีต้องใช้ฟังก์ชันเฉพาะ ให้เลือก Sign in / Login / Account
  5. ยืนยันว่าเข้าสู่ระบบสำเร็จจากสถานะบัญชีบนหน้าจอ

ผลลัพธ์ที่คาดหวัง

  • ครูสามารถเปิดหน้า Dashboard ได้และเริ่มสำรวจข้อมูลได้ทันที
  • ครูแยกได้ว่าเมื่อไรควรใช้ Public View และเมื่อไรควร Login
