เว็บไซต์คู่มือ **SatFinder Manual** จัดทำขึ้นเพื่อเป็นเอกสารสำหรับครูผู้สอนในการใช้งาน TinyGS Dashboard และการเรียนรู้ด้านโทรคมนาคมผ่านดาวเทียมสำหรับโรงเรียนทั่วประเทศ ภายใต้โครงการ SatFinder

## ผู้จัดทำ
คณะวิศวกรรมศาสตร์ มหาวิทยาลัยธรรมศาสตร์ และ สถาบันวิจัยและให้คำปรึกษาแห่งมหาวิทยาลัยธรรมศาสตร์

## หน่วยงานสนับสนุน
กองทุนวิจัยและพัฒนากิจการกระจายเสียง กิจการโทรทัศน์ และกิจการโทรคมนาคมเพื่อประโยชน์สาธารณะ
