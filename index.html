SatFinder x TinyGS

คู่มือการใช้งาน TinyGS Dashboard สำหรับครูผู้สอน

เอกสารประกอบการอบรมโครงการส่งเสริมการเรียนรู้ทางด้านโทรคมนาคมในโรงเรียนทั่วประเทศ (SatFinder) โดยคณะวิศวกรรมศาสตร์ มหาวิทยาลัยธรรมศาสตร์ และสถาบันวิจัยและให้คำปรึกษาแห่งมหาวิทยาลัยธรรมศาสตร์ สนับสนุนโดย กสทช. เพื่อประโยชน์สาธารณะ

วัตถุประสงค์คู่มือ

  • ช่วยครูเข้าใจการใช้งาน TinyGS Dashboard อย่างเป็นลำดับ
  • เตรียมเนื้อหาสำหรับขยายผลให้กับนักเรียนที่โรงเรียน
  • ใช้ภาพหน้าจอจริงเพื่อให้สอนได้ง่ายและเห็นภาพชัดเจน

แนวทางใช้งานคู่มือฉบับนี้

1) อ่านภาพรวมก่อนสอน

เริ่มจากเข้าใจโครงสร้างของ TinyGS Dashboard และเมนูหลักก่อนลงรายละเอียดแต่ละหน้า

2) ใช้ภาพหน้าจอประกอบ

แต่ละหัวข้อมีตำแหน่งวางภาพ Print Screen เพื่อให้ผู้เรียนเชื่อมโยงกับหน้าจอจริงได้ทันที

3) ดัดแปลงกิจกรรมได้

สามารถปรับบทเรียนเป็นกิจกรรมในชั้นเรียน เช่น ฝึกอ่านข้อมูลดาวเทียมและสัญญาณจริง

โครงร่างเนื้อหาคู่มือ TinyGS Dashboard

รองรับการเติมภาพหน้าจอจริงทีละส่วนเมื่อได้รับไฟล์จากทีมงาน

  • แนะนำ TinyGS คืออะไร และใช้ทำอะไรในงานการเรียนการสอน
  • ขั้นตอนเข้าหน้าเว็บไซต์และลงชื่อเข้าใช้
  • คำอธิบายเมนูหลักที่ควรรู้ก่อนเริ่มใช้งานจริง
พื้นที่วางภาพ: หน้า Login / หน้าแรกหลังเข้าสู่ระบบ

  • อ่านข้อมูลสถานี (Ground Station) ที่เชื่อมต่อกับระบบ
  • ตรวจสอบสถานะออนไลน์, กราฟ และค่าพื้นฐานที่สำคัญ
  • แนวทางตีความข้อมูลเพื่ออธิบายให้นักเรียนเข้าใจได้ง่าย
พื้นที่วางภาพ: หน้า Station Overview / สถานะออนไลน์

  • ดูรายการดาวเทียมที่สนใจและช่วงเวลาที่คาดว่าจะรับสัญญาณได้
  • อธิบายข้อมูลการรับสัญญาณ (เช่น packet, timestamp, frequency)
  • ตัวอย่างคำถามชวนคิดให้นักเรียนวิเคราะห์ข้อมูลจริง
พื้นที่วางภาพ: หน้า Satellite Passes / Packets Received

  • ตัวอย่างกิจกรรม 15–30 นาทีที่ทำได้ทันที
  • ใบงานสั้น ๆ เพื่อฝึกอ่านค่าจาก Dashboard
  • เชื่อมโยงบทเรียนกับหลักการโทรคมนาคมและอวกาศ
พื้นที่วางภาพ: ตัวอย่างกิจกรรม / ใบงานประกอบ

หมายเหตุการเตรียมภาพประกอบ

เมื่อได้รับไฟล์ภาพ Print Screen หน้าจอ TinyGS เพิ่มเติม สามารถแทนที่กล่อง “พื้นที่วางภาพ” ในแต่ละส่วนได้ทันที เพื่อให้คู่มือพร้อมเผยแพร่บน GitHub Pages อย่างเป็นทางการ
